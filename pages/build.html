Building Naijav

One joy of the Naijav is that it is all Open Source. So you get to clone the repo, type in some commands and watch the App build. Sounds so awesome of course. It allows you to swap that background color you do not like with your favorite.

For such reasons, dear to all of us, instructions to build the application have been made available.

Prerequisites #

So what do we require to build Naijav?

  • Ionic
  • Git

Ensure you have installed these requirements as we will use then in the next steps.

Download and Build #

First, clone the Github repo.

⇒ git clone https://github.com/forfuture-dev/naijav.git

Once that is finished. You need to add the platform you want to build Naijav for. Make sure you have all thre requirements met for the platform you want to build for. See ionic's documentaion on such requirements

⇒ ionic platform add android # for android
⇒ ionic platform add ios # for ios
⇒ ionic platform add  # replace  with your target platform

Installation #

Now we at the good part: Building! Now type this magical command:

⇒ ionic build  # replace  with android, ios, etc.
